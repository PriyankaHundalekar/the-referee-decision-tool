# The Referee - Decision Making Tool

A smart comparison tool that helps users make informed decisions by analyzing trade-offs between different options instead of providing single answers.

## Features

- Compare APIs, cloud services, and tech stacks
- Structured pros/cons analysis
- Constraint-based recommendations
- Interactive comparison interface
- Export comparison reports

## Tech Stack

- Frontend: React with TypeScript
- Backend: Node.js with Express
- Styling: Tailwind CSS
- Deployment: Ready for AWS deployment

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
├── src/
│   ├── components/     # React components
│   ├── services/       # API and comparison logic
│   ├── types/          # TypeScript definitions
│   └── utils/          # Helper functions
├── public/             # Static assets
└── .kiro/              # Kiro configuration
```

## Usage

1. Select comparison type (API, Cloud Service, Tech Stack)
2. Input your requirements and constraints
3. Get detailed comparison with pros/cons
4. Make informed decisions based on trade-offs
